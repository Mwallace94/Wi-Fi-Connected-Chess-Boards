--------------------------------------------------------------------------------
                        ESP8266 Breadboard Color Code
--------------------------------------------------------------------------------

green       : rst
red         : txd
grey        : rxd
orange      : gpio0
brown       : gpio2
white       : ch_pd

--------------------------------------------------------------------------------
                            ESP8266 Programming Guide
--------------------------------------------------------------------------------

    Connect gpio0 to ground and reset module to enter bootloader mode. Put bus pirate into transparent UART mode. Use arduino IDE or ESP8266 flash download tool to install bin image into ESP flash.
    
--------------------------------------------------------------------------------
                            Bus Pirate Transparent UART Mode
--------------------------------------------------------------------------------

    Plug in bus pirate and connect using serial terminal at correct com port and 115200 baud. Type into serial port the following commands, each followed by enter, and do not type parenthesis: "m", "3", "9", "1", "1", "1", "2", "W", "(1)", "y". The device should now be in transparent UART mode with a baud rate of 115200.
